SketchWorld

A Mobile-Friendly Sharing-Focused Creative Coding Platform for Schools, Students, Artists, and Enthusiasts
Motivation

Running CoderDojo afterschool programs around Dartmouth, I noticed that kids seemed to get bored of Scratch after a while. Many of them expressed a desire to move on and start learning "real" coding.


Vision

Design & implement a coding learning environment with the following key features:

  • Online, mobile-friendly interface, so that students can access the site from anywhere, on any device.
  • Easy "click-and-run" interface, so that creators don't get trapped in the wild west of setting up a coding environment.
  • An accessible but powerful graphics library, allowing beginners' first creations to be far more engaging that a command-line hello world.
  • Viral sharing potential through features that allow kid's sketches (mini-programs) to be shared like links, installed like apps, and run like games.
